## About
Wayang App is a wayang classification and information application. This application provides information about wayang as an Indonesian culture, with a focus on the categories of Wayang Beber, Wayang Gedog, Wayang Golek, Wayang Krucil, Wayang Kulit, and Wayang Suluh. This application is an interactive and innovative guide for wayang enthusiasts, students of traditional arts, and anyone wishing to explore Indonesia's rich heritage. 

This application is also part of the final project "APPLICATION OF CONVOLUTIONAL NEURAL NETWORK METHOD IN THE DEVELOPMENT OF AN ANDROID-BASED WAYANG TYPE CLASSIFICATION APPLICATION" at Gunadarma University.

**Features**
1. Wayang menu, to see detailed information about all types of wayang
2. Classify the types of puppets using images from the camera or gallery on your smartphone.
3. Video about basic wayang technique. By [YT Animasi Emon](https://youtu.be/b8mZRqU8CsU?si=muv4LlXFG-wlt8Jc)
4. Additional menus, information on puppeteers, museums and studios
5. Use of classification features without internet

**Technology**
* Android Studio IDE to develop the Android application.
* Tensorflow and Keras to develop, train, and deploy ML model.
